# Movila Backend

This is the backend API for the Movila project.

## Tech Stack
- Node.js
- Express
- TypeScript

## Setup
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```

## Project Structure
- `src/` — Source code
- `uploads/` — File uploads

## Environment Variables
Create a `.env` file in the backend directory with your configuration.
